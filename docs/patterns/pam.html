PAM Admin Interface Page Catalogue


Pages consist of everything in the browser window excluding the left and top menu bars. The PAM Admin Interface consists of several pages, each using one of 6 different patterns.

Patterns are re-used throughout the product. Using the established patterns will help users learn the product and keep maintaining the product simpler and more time-efficient.


Dashboard page


A dashboard page is comprised of one or more card components.


List page


A list page serves as a directory for all like-assets, such as users, user groups, devices, etc.


Show page


An show page is for more detailed information on an asset such as a specific user, user group or device.


System configuration page


A system configuration page consists of one or more tabs. Each tab represents a category of settings. Each tabbed frame then utilises a tree structure to list configurable elements.
